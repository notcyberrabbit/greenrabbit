# 🚀 GreenRabbit Quick Start Guide

## 5-Minute Setup

### Step 1: Clone & Install
```bash
git clone <your-repo>
cd greenrabbit
npm install
```

### Step 2: Get API Keys
1. **Anthropic API Key** (Required)
   - Go to https://console.anthropic.com
   - Create a new API key
   - Copy it

2. **Bags.fm API Key** (Optional - demo uses mock data)
   - Go to https://bags.fm/api
   - Generate an API key (if available)
   - Copy it

### Step 3: Configure Environment
```bash
cp .env.example .env.local
```

Edit `.env.local`:
```
ANTHROPIC_API_KEY=sk-ant-... (paste your key)
NEXT_PUBLIC_BAGS_API_KEY=... (optional)
NEXT_PUBLIC_BAGS_API_URL=https://api.bags.fm
```

### Step 4: Run
```bash
npm run dev
```

Visit http://localhost:3000 and test with any Solana wallet address!

## Deploy to Vercel (1 minute)

1. Push code to GitHub
2. Go to https://vercel.com
3. Click "New Project" → Select your repo
4. Add environment variables:
   - `ANTHROPIC_API_KEY` = your key
   - `NEXT_PUBLIC_BAGS_API_KEY` = your key (optional)
5. Click "Deploy"
6. Done! Your app is live 🎉

## Common Issues & Fixes

### Issue: "Invalid wallet address"
**Fix:** Enter a valid Solana address (44 chars, starts with 4-5 chars, base58)
Example: `98vb2QQonCHYw9wy8DtqKZDR9...`

### Issue: "Failed to fetch wallet data"
**Fix:** 
1. Check Bags API key in `.env.local`
2. Verify wallet has Bags tokens
3. (App uses demo data if API fails)

### Issue: "Failed to analyze wallet"
**Fix:**
1. Check `ANTHROPIC_API_KEY` is correct
2. Check you have API credits
3. Check rate limits: https://console.anthropic.com

### Issue: Styles look broken
**Fix:**
1. Hard refresh browser: Ctrl+Shift+R
2. Delete `.next` folder: `rm -rf .next`
3. Rebuild: `npm run dev`
